# Changelog

All notable changes to the BULK HOMME Magento 2 Theme will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11-28

### Added
- Initial release of BULK HOMME theme
- Minimalist black and white design system
- Fully responsive layout (mobile, tablet, desktop)
- Complete color palette with 10 grey shades
- Typography system with 5 heading levels
- Custom button styles (primary, secondary, text)
- Form elements with multiple states
- Product card components with hover effects
- Mobile navigation with hamburger menu
- Custom header and footer templates
- Product page customization
- Shopping cart styling
- LESS variables system for easy customization
- Reusable mixins library
- Comprehensive documentation (README, INSTALLATION, CHANGELOG)
- Support for Magento 2.4.x
- Performance optimizations
- Accessibility features (WCAG 2.1 Level AA)
- SEO-friendly structure
- Print styles
- Loading skeleton animations
- Focus states for keyboard navigation
- Custom grid system
- Spacing scale (8px base)
- Shadow elevation system
- Smooth transitions and animations
- Cross-browser compatibility

### Design Features
- Sharp corners (0 border-radius) for modern look
- High contrast typography
- Generous whitespace
- Premium hover effects
- Consistent spacing system
- Responsive font scaling
- Mobile-first approach

### Components Included
- Buttons (3 variants)
- Input fields (text, email, password, tel, number)
- Textarea
- Select dropdown
- Checkbox
- Radio buttons
- Toggle switch
- Product cards
- Navigation menu
- Header
- Footer
- Forms
- Cards and containers

### Browser Support
- Chrome (latest 2 versions)

- Chrome Android (latest 2 versions)

### Performance

- Optimized images

### Documentation
